# generateFlutterPage Skill

## Purpose
Generate a Flutter page scaffold with state and navigation hooks.

## Required Inputs
- Page name
- State requirements
- Navigation route

## Outputs
- Dart page widget scaffold

## Constraints
- Null-safety required.

## Example
Input: `ProfilePage`. Output: `profile_page.dart` with app bar and state placeholders.
