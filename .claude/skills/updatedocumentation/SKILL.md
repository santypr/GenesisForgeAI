# updateDocumentation Skill

## Purpose
Apply incremental documentation updates after scope/code changes.

## Required Inputs
- Change summary
- Current docs snapshot

## Outputs
- Updated markdown files and changelog entries

## Constraints
- Preserve existing headings and historical records.

## Example
Input: added new API endpoints. Output: updated technical analysis, test cases, changelog.
