# generateArchitectureDiagram Skill

## Purpose
Generate architecture diagrams in Mermaid or ASCII form.

## Required Inputs
- Components
- Dependencies
- Data flow notes

## Outputs
- Diagram block for `/docs/architecture/diagrams.md`

## Constraints
- Prefer Mermaid when target renderer supports it.

## Example
Input: React + API + DB topology. Output: Mermaid graph TD diagram.
